## 📝 Sobre o Projeto

Um clássico jogo de **Pedra, Papel e Tesoura** desenvolvido em Java para rodar diretamente no terminal. O projeto utiliza lógica condicional para determinar o vencedor e interage com o usuário através de entradas via teclado.

Ideal para estudos sobre:
* Programação Orientada a Objetos (POO).
* Manipulação de entradas com `Scanner`.
* Geração de números aleatórios com `Random`.
* Estruturas de repetição (`while`).

---

## 🎮 Como Funciona

O jogo segue as regras tradicionais:
1. **Pedra** vence **Tesoura**.
2. **Tesoura** vence **Papel**.
3. **Papel** vence **Pedra**.

### 🛠️ Funcionalidades
- [x] Escolha intuitiva por menu numérico.
- [x] Oponente controlado por Inteligência Artificial (aleatória).
- [x] Validação de jogadas (evita números inválidos).
- [x] Opção de "Jogar Novamente" sem fechar o programa.

## 🚀 Como Executar

1. Certifique-se de ter o **JDK** instalado em sua máquina.
2. Clone este repositório:
   ```bash
   git clone [[https://github.com/alvesthzin-art/nome-do-repositorio.git](https://github.com/alvesthzin-art/nome-do-repositorio.git](https://github.com/alvesthzin-art/Jokenpo.git))
Compile o arquivo:

Bash
javac Jokenpo.java
Execute o jogo:

Bash
java Jokenpo
